VSH服 藤原妹红 飘上月球，不死之烟

六月 23, 2023
  • -TF2

新BOSS - 藤原妹红

生生生生暗生始，死死死死冥死终。

  • 火焰超级跳：超级跳时，会短暂点燃太靠近的敌人。超级跳的冷却时长比一般BOSS要久。会有一个火圈特效。
  • 近战攻击伤害一般，但是对点燃的敌人有爆击。
  • 有一个彩虹喷灯，伤害低，没后续燃烧伤害，用于点燃敌人。
  • 因为不老不死的特性，在游戏中可抗拒一次满目疮痍，获得继续战斗的能力，恢复的血量根据剩余人数计算。
  • 自灭火焰大旋风：在地面时激活愤怒技能，会短暂充能后释放，点燃大范围1000单位内的敌人，同时对于太靠近500单位的敌人，还会附加爆炸。
  • 不死鸟的超高温羽毛：在浮空时激活愤怒技能，会产生一个持续20秒的燃烧立场，靠近的敌人就会被瞬间点燃。自己获得无击退，防御，加速加成。

特殊回合更新

  • APose：平民模式，大家都是良民。
  • 逮虾户：全部人都成为老司机。

BOSS更新

终结者

  • 武器改为沙包说很土的短路发生器。
  • 增加检查是否为幽灵猎手死亡，这样子死掉的幽灵猎手不会变成小红。
  • 不论变成小红还是幽灵猎手均无超量治疗。

B_S_Brony

取消迷你步哨建造，改为普通步哨。

德克萨斯

降低剑雨高度。

红队更新

侦察兵

增加猎枪子弹。

机枪手

开局一小段时间会获得速度加成，方便速度缓慢的胖子到达想要的位置。

工程师

开局一小段时间会获得建造速度加成，免得你什么也没干就被打死了。

狙击手

弓箭的伤害太超模了，射得BOSS连妈妈都不知道是谁，削弱10%伤害。

修复

特殊回合：避免救世主模式产生Invalid memory access错误；使用一种更加安全的办法在回合结束时删除掉所有人的钩爪。

DIO: 放弃使用 tank_destruction 实体作为爆炸效果，而转为使用 env_shake 和 info_particle_system 来表现震动和爆炸粒子效果。
