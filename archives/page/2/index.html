FF2服 7.16日更新 - 慢动作攻击/子弹时间
2020年07月16日 527 字 大概 2 分钟

首先总结一下慢动作攻击/子弹时间为什么会卡，写在这里，以便其他人参考。

慢动作攻击需要更改服务器的Cvar host_timescale 为对应的参数，但此命令同时需要服务器更改sv_cheats为1，也就是开启作弊。这个本是一个作弊指令。

插件在执行慢动作攻击的时候，先把sv_cheats设为1，再更改 host_timescale ，但可能处于服务器保护或者其他的原因，这个更改没有完成，我需要做的就是把sv_cheats更改为1，但是这是一个很危险的举动，因为玩家可能会用其来作弊。于是通过Cheat Control插件的限制，普通玩家将不能使用任何的作弊指令，这样就解决了慢动作攻击的问题。希望以后原作者能做到既不打开作弊又能更改 host_timescale 。

  • 更改 十六夜 咲夜 的慢动作攻击持续时间为6.0秒
  • 更改 弗拉基米尔 普京 的慢动作攻击持续时间为3.0秒
  • 更改 博丽 灵梦 的慢动作攻击持续时间为4.0秒
  • 更改 团队杀手 的慢动作攻击持续时间为5.0秒
  • 更改 魂魄妖梦 的慢动作攻击持续时间为5.0秒
  • 更改 射命 丸文 的慢动作攻击持续时间为5.0秒
  • 更改 黑岩★射手 的慢动作攻击持续时间为3.0秒
  • 更改 射命 丸文 的慢动作攻击持续时间为5.0秒
  • 更改 G胖 的慢动作攻击持续时间为3.0秒
  • 更改 杰哥 的慢动作攻击持续时间为4.0秒
  • 更改 MLG狙击手 的慢动作攻击持续时间为3.0秒
  • 更改 忍者间谍 的慢动作攻击持续时间为4.2秒

补充一个改动，Boss vs Boss模式有10%的概率触发。目标为击杀所有的敌人，由于使用了新的眩晕插件，应该能够区分敌我的眩晕。但未经测试，听天命吧。

计划给FF2服的大修大改
2020年07月08日 658 字 大概 2 分钟

此为7月7日对FF2服进行的大修记录。因为我参加了FreakFortressBat的翻译。以后服务器的数据会以此为准，再在需要的基础上进行修改。在大修部署的初期，会根据玩家们给我的意见，进行进一步的修改。

这个更新尚未实装，主要是为了解决服务器目前已有的问题。

此更新已在2020/7/11 3:34AM 实装

原Github: https://github.com/Batfoxkid/FreakFortressBat/
下面是一些需要获得注意的改动，具体的情况请在游戏中体验。

通用：
- 兵种限制将被取消
- Boss的生命值和愤怒值将回调到数据膨胀之前
- Boss基础伤害稍微降低
- 对Boss数量进行调整
- 武器数值以FreakFortressBat的标准进行微调
- 对Boss的描述进行完全翻译
- 修复慢动作攻击(子弹时间)卡成狗的问题和其他问题

侦察兵：
- 所有武器的子弹数量增加
- 睡魔切出速度减慢，仍然可以眩晕Boss
- 睡魔和包装纸杀手的球的数量减少

士兵：
- 武器的子弹数量增加
- 火箭伞兵有独特的属性，火箭跳的时候伤害极小，便于跳跃

火焰兵：
- 不能再使用火焰喷射器进行超量治疗
- 火焰喷射器子弹数量增加
- 偷袭火焰不再有逆天的推力
- 龙之怒焰不再有命中增加弹药的属性

爆破手：
- 武器的子弹数量增加
- 没有命中回血的属性了
- 粘弹跳跃者有独特的属性，粘弹跳的时候伤害极小，便于跳跃

机枪手：
- 武器的子弹数量增加
- 娜塔莎不能减速Boss，但是在告诉转动时仍能保持较快的移动速度
- 没有自伤跳的手段
- 钢铁拳套也可以抵挡近战伤害

工程师：
- 传送扳手恢复使用，但是不能够随便在重生点和传送出口之间传送
- 获得传送装置的改造，跳板和加速
- 限制了全队的步哨数量，普通步哨不能超过4个，小步哨不能超过4个 总步哨数量不能超过6个
- 没有自伤跳的手段

狙击手：
- 没有自伤跳的手段
- 近战攻击墙面可以爬墙，需要扣除生命值
- 弓箭可以爆炸

医生：
- 所有武器的子弹数量增加
- 注射枪不需要装填
- 十字弩可以爆炸
- 注射枪可以使用针筒跳

间谍：
- 可以使用隐形刺客了
